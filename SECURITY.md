# Security Policy

Echo is an MVP desktop application for local Docker workflows.

## Reporting a Vulnerability

Please do not report security issues in public issues. Use GitHub's private vulnerability reporting for this repository when available, or contact the maintainers privately through GitHub.

Include:

- The affected Echo version or commit.
- The host operating system.
- Steps to reproduce.
- Impact and any known workaround.

We will acknowledge valid reports as soon as practical and prioritize fixes based on severity.
